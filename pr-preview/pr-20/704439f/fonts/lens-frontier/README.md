# Lens Frontier self-hosted fonts

Generated from the Google Fonts CSS API request used by the site:

[Google Fonts CSS API request](https://fonts.googleapis.com/css2?family=IBM+Plex+Mono:wght@400;500;600&family=IBM+Plex+Sans:wght@400;500;600;700&family=IBM+Plex+Serif:wght@500;600;700&family=Noto+Serif+SC:wght@500;700&display=swap)
