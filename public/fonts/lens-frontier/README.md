# Lens Frontier self-hosted fonts

Generated from the Google Fonts CSS API request used by the site:

[Google Fonts CSS API request](https://fonts.googleapis.com/css2?family=IBM+Plex+Mono:wght@400;500;600&family=IBM+Plex+Sans:wght@400;500;600;700&family=IBM+Plex+Serif:wght@500;600;700&display=swap)

The site self-hosts the Latin WOFF2 files to avoid blocking first visits on remote Google Fonts CSS. The stylesheet is still loaded after first render by the page layout. Chinese text uses system serif fallbacks such as Songti SC instead of self-hosted CJK webfont shards.
